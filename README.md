# emailApp

**Description**: This is simple emailing application.

### How it works: 
 1. Enter the email address you want to send email to.
 2. Enter  the email subject
 3. Enter the email
 4. Attach a single file
 5  Click on the send button.
 
### Major technologies:
 - NodeJS
 - Express.js
 - Nodemailer
 - Googleapis
 - Multer
